# 担当箇所
・ウイルスの基礎的な移動(その後別メンバーが改善)<br>
・アイテムを使えるようにする処理の追加<br>
・ウイルスがゴールしたらクリア画面に映る処理<br>
・ゴールしたら次のステージに進む処理<br>
・リトライ機能<br>
・ウイルスが通れなくなったら点滅させ、再度リトライ(その後アニメーションに仕様変更)<br>
・ステージの追加<br>
・クリアとクレジットの基礎的な部分の実装(画像処理などは別メンバーが実装)<br>

# ゲーム概要
地形を上書きしてウイルスを誘導し、ゴールを目指す2Dパズルゲーム<br>
ウイルスは黄色い線に沿って移動する<br>
線の外に出るとゲームオーバー<br>
光る線のアイテムを獲得し、線の上でボタンを押すと道が上書きされる<br>
全11ステージ<br>
